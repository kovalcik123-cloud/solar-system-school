# solar-system-school
Interactive 3D Solar System explorer designed for Slovak elementary schools
Slnečná Sústava 3D (Školská Edícia)

Interactive 3D Solar System explorer designed for Slovak elementary schools.

Features

Real-time 3D Rendering: Uses Three.js for smooth performance on school tablets and PCs.

Accurate Data: Planet sizes, distances (scaled), and orbital periods.

Interactive Quiz: 8-question test with instant feedback and a final results certificate.

Gatekeeper: Password protected access (Code: NGV-2025).

Touch & Mouse Support: Fully responsive for interactive whiteboards and tablets.

How to Run

Open the Live Demo.

Enter the class access code.

Enjoy exploring!
